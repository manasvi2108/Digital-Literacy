The main focus of this folder is on identifying and preventing digital financial risks, i.e., "UPI" and "Online Payment Fraud," which is a high-risk area for college students in India.

Inside This Folder

casestudy.md: A detailed casestudy on "Request Money" UPI Scam and its consequences.

prevention-checklist.md: A list of 8 safety tips for preventing UPI scams, including official reporting procedures (Helpline - 1930).

Key Takeaway

The key takeaway from this folder is, "A UPI PIN is required when sending money, but it is NOT required when receiving money. Be cautious and immediately report suspicious activity to cybercrime.gov.in."
